# Airline Ticket Price Prediction

A Machine Learning based Airline Ticket Price Prediction system built using Python, Flask, XGBoost, Pandas, and Scikit-learn.

The project predicts flight ticket prices based on:
- Airline
- Source
- Destination
- Journey Date
- Travel Class

## Technologies Used
- Python
- Flask
- XGBoost
- Pandas
- NumPy
- Scikit-learn
- HTML/CSS

## Features
- Interactive web interface
- ML-based ticket price prediction
- Multiple airline and route support
- Travel class based pricing
- Real-time prediction system

## How to Run

1. Clone the repository
2. Install required libraries

```bash
pip install flask pandas numpy scikit-learn xgboost
```
